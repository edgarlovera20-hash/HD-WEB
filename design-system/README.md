# Heavenly Dreams Design System

This system standardizes the Heavenly Dreams digital ecosystem with one visual language for the public website, RH recruiting app, and administration app.

## Tokens

- Colors: corporate blue, cyan accent, dark surfaces, semantic success/warning/error.
- Typography: Inter for interface text and Poppins for display headings.
- Radius: 8, 12, 16, 20, and 24px.
- Motion: CSS transitions or Framer Motion at 150ms, 250ms, and 350ms.
- Icons: Lucide Icons only.

## Components

Use the shared classes in `components/components.css` for buttons, cards, forms, widgets, alerts, tables, modals, and AI chat bubbles.
