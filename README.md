# Heavenly Dreams Web 2026

Prototype web design for Heavenly Dreams SAS de CV.

## Run locally

```bash
node server.cjs
```

Open:

```text
http://127.0.0.1:4173/
```

## Included

- Corporate landing page
- About, mission, vision and values
- Partner brands carousel
- Vacancies dashboard
- Candidate form
- Recruitment chatbot mock
- Culture and contact sections
